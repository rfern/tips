Tips, Tips and, um, maybe some more Tips!

We always have things that we think everyone should know. Like that fancy new tool you discovered that lets you build a responsive website in 3 seconds. Or that wonderful git command that lets you fix everything (maybe in a fit of rage...).

But how do you let everyone know? Standing on the highest rooftop and shouting out unfortunately requires people to be close enough to hear you. Luckily, with the recent advent of this thing called 'The Internet', we have a way to solve the problem.

Welcome to the UQ Computing Society Tips site.

This is your place to share tips with everyone in an easy and open way. The best part is getting involved is very simple, and you can use the tips on this site to learn how to do it!

How to add your tips (or make some improvements to the site)

  1. Fork the repository
  2. Clone your repo to your computer
  3. Modify the relevant page or add a new one
  4. Commit your changes and push it up
  5. Open a Pull Request against the original repository
  6. Keep an eye on the Tips site for your changes to appear!
  7. For more information how to use git please see https://try.github.io/
